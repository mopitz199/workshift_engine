Assignation Proxy¶

class proxies.assignation_proxy.AssignationProxy(obj, *args, **kwargs)¶

This class is to map the attributes of an assignation and their relations. Some requirements from the assignation are:

  • id: The id of the assignation (if is created in the database)

  • starting_date: The starting date of the assignation

  • ending_date: The ending date of the assignation

  • workshift_id: The workshift id of the assignation

  • workshift: The workshift obj

  • person_id: The person if of the assignation

  • person: The person obj

  • starting_day: The starting day of the assignation

get_differences()¶

This function return the difference of range between the init range and the current range
